N명이 먹기에 적당한 치킨은 몇마리인가?

남기지 않는다. 치킨. 피보나치킨:
https://fibonachicken.herokuapp.com/

## 피보나치킨 알고리즘 ##
N이 주어질 때 피보나치킨 수를 구하라!

N이 피보나치 수이면 O(1)에, 아니면 O(k)에 가능.
여기서 k는 N보다 작은 가장 큰 피보나치 수가 k번째 피보나치 수임.
